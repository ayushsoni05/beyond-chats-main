# React Frontend

Displays articles from the Laravel API (original and updated).

## Run
1) `npm install`
2) Copy `.env.example` to `.env` and set `VITE_API_BASE` (default http://localhost:8000/api)
3) `npm run dev` and open the printed URL.

## Features
- Responsive grid of article cards
- Shows status (Original vs Updated)
- Renders body and citations list
